# ikonqu (Twinpath Icons) - v5.15.3 [Legacy]

Koleksi ikon FontAwesome Pro v5 yang dikelola secara internal oleh Twinpath. Versi ini adalah versi **Legacy** yang tetap dipertahankan untuk kompatibilitas proyek lama.

## Instalasi

### Dari npm (Registry)
```bash
npm install @twinpath/ikonqu@5
```

### Dari GitHub (Direct)
```bash
npm install twinpath/ikonqu#v5
```

## Cara Penggunaan
Cukup import di entry point proyek Anda:

```javascript
import "@twinpath/ikonqu";
```

## Versi Terbaru
Jika Anda ingin menggunakan fitur terbaru dari FontAwesome v6, silakan gunakan branch **main**:
*   **Branch main**: [FontAwesome 6.0.0](https://github.com/twinpath/ikonqu/tree/main)
*   Instalasi v6: `npm install @twinpath/ikonqu` atau `npm install twinpath/ikonqu`
